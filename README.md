# 📰 Fake News Detection using NLP & ML

## Overview
A machine learning project to detect fake news articles using 
NLP techniques and classification algorithms.

## Dataset
- True.csv — Real news articles
- Fake.csv — Fake news articles
- Source: Kaggle - Fake and Real News Dataset

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Models Used
|          Model           | Accuracy |
| Naïve Bayes (TF-IDF)     |   ~94%   |
| SVM - LinearSVC (TF-IDF) |   ~99%   |

## How to Run
1. Install dependencies:
pip install pandas numpy scikit-learn matplotlib seaborn

2. Open Jupyter Notebook:
jupyter notebook fake_news_detection.ipynb

3. Run all cells in order

## Project Structure
- fake_news_detection.ipynb — Main notebook
- True.csv — Real news dataset
- Fake.csv — Fake news dataset  
- manual_testing.csv — Manual test cases
